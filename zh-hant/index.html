如何兌換禮包碼

依照這些簡單步驟來領取您的獎勵。

  1. 1
    打開《Wittle Defender》遊戲，點擊左上角的 個人頭像。
  2. 2
    在個人資料頁面，點擊您 玩家ID 旁邊的「複製」按鈕。
  3. 3
    前往官方兌換中心：前往兌換中心
  4. 4
    依序貼上您的玩家ID、想兌換的禮包碼，並輸入頁面上的驗證碼。
  5. 5
    點擊「兌換 (Redeem)」，然後返回遊戲，在您的遊戲內信箱中領取獎勵！

Wittle Defender 英雄強度排名 (2025年6月)

最權威的英雄排名，幫助您打造最佳隊伍。

級別英雄
S劍聖, 機器人, 神之統治者
A惡魔之子, 火焰女巫, 劍術大師, 烈焰射手, 冰魔, 冰之女巫
B貓刺客, 暗夜男爵, 雷電法老, 火焰法師, 大祭司
C惡魔獵手, 冰法師
D科學怪人, 冰狼幼崽, 及其他史詩/普通單位。

神級 (S Tier)

劍聖

  • + 無需全力升級即可造成高額傷害。
  • + 優秀的鎖定機制，便於集火Boss。

神之統治者

  • + 遊戲中潛在傷害最高。
  • + 巨量的範圍傷害，清場能力極強。

機器人

  • + 傷害輸出名列第二。
  • + 可被培養成強大的前排坦克。

頂級強力 (A Tier)

惡魔之子

  • + 對Boss單體傷害極高。
  • - 依賴運氣獲得好的升級。

火焰女巫

  • + 對付大量敵人時傷害很高。
  • - 技能冷卻時間長。

劍術大師

  • + 非常適合清理大量小怪。
  • - 對付Boss時表現不佳。

烈焰射手

  • + 能召喚單位提供功能性。

冰魔

  • + 召喚單位分散敵方注意。
  • - 依賴隊友提供主要輸出。

冰之女巫

  • + 可冰凍敵人進行控場。
  • - 極度依賴升級。

堪用 (B Tier)

貓刺客

  • + 單體傷害不錯。
  • - 升級不值得投資。

暗夜男爵

  • + 主要價值在於連鎖技能。
  • - 單獨作戰能力不強。

雷電法老

  • + 與機器人搭配良好。
  • - 初始傷害低。

情境角 (C Tier)

這些單位在沒有強大隊伍的情況下表現不佳。最好盡快找到更好的選擇來替換他們。

惡魔獵手

  • + 「散射」升級有助於範圍傷害。
  • - 即使滿級傷害也很低。

冰法師

  • + 與冰之女巫的連鎖技能很強。
  • - 單獨作戰能力因低傷害而不足。

不建議使用 (D Tier)

這些後衛無法為隊伍提供任何價值，因為他們的屬性、升級和傷害都很低。

科學怪人

  • - 傷害低，技能無用。

冰狼幼崽

  • - 因低傷害很快就變得無用。
